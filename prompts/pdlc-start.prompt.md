# Command: /pdlc-start

**When to use**: To initiate a brand new feature or bug fix.

**Inputs**:
- User request: [Clear description of WHAT is needed]

**Stages involved**:
- Problem Decomposer
- Planner

**Artifacts**:
- `requirements.md`
- `plan.md`

**Example**:
`/pdlc-start "Add a dark mode toggle to the settings page"`
